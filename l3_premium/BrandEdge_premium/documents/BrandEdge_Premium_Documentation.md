## 系統概述

BrandEdge 旗艦版是一個企業級品牌印記引擎，透過AI深度分析顧客評論，提供全方位的品牌策略建議。系統整合OpenAI GPT-4o-mini進行智慧化文本分析，支援10-30個產品屬性的多維度分析。

### 核心特色
- **擴展屬性分析**：支援10-30個產品屬性（相較於標準版6個）
- **大規模資料處理**：最多1000則評論，20個品牌同時分析
- **八大分析模組**：完整覆蓋從市場輪廓到品牌識別度的策略分析（已移除PCA功能）
- **雙資料庫架構**：PostgreSQL生產環境，SQLite開發測試
- **集中式管理**：Prompt和Hint統一管理，便於維護更新
- **完全動態化分析**：所有分析結果基於實際資料計算，無硬編碼內容
- **智慧命名系統**：三層備用機制確保穩定運行

### 版本資訊
- **當前版本**：v3.2.1 Flagship Edition
- **更新日期**：2025-01-12
- **最新改進**：
  - 移除PCA定位地圖功能
  - 更新系統名稱為「品牌印記引擎」
  - 統一聯絡資訊為 partners@peakedges.com
  - 移除測試帳號顯示以提升安全性

## 功能模組詳解

### 現有八大分析模組

#### 1. 增強版市場輪廓模組（marketProfileEnhancedModule）
- **功能**：市場區隔分析、AI動態命名、客群偏好分析
- **技術**：K-means聚類、智慧命名系統、三層備用機制

#### 2. 市場賽道分析模組（marketTrackModule）
- **功能**：識別高成長潛力屬性、計算市場機會分數、四象限策略分析
- **技術**：成長潛力計算、重要性權重分析、動態視覺化
- **核心指標**：
  - **成長潛力**：衡量從目前表現提升到理想水準的改善空間（0-100%）
  - **市場重要性**：反映屬性對整體市場競爭力的影響程度（0-100%）
  - **賽道得分**：綜合成長潛力與重要性的機會評分
- **四象限策略矩陣**：
  - **高潛力賽道**（右上）：高成長潛力+高重要性 → 最優先投資領域
  - **成熟賽道**（左上）：低成長潛力+高重要性 → 維持領先優勢
  - **利基賽道**（右下）：高成長潛力+低重要性 → 差異化機會
  - **低優先賽道**（左下）：低成長潛力+低重要性 → 資源最低配置
- **視覺化呈現**：
  - 散點圖展示各屬性在成長潛力-重要性矩陣中的位置
  - 點的大小代表賽道得分（機會大小）
  - 顏色深淺表示優先級高低
- **策略建議**：自動生成前5大優先發展賽道，並提供具體數據支持

#### 3. 進階DNA分析模組（advancedDNAModule）
- **功能**：品牌DNA雷達圖、競爭力評估
- **技術**：多維度視覺化、相對表現分析

#### 4. 關鍵因素分析模組（keyFactorsModule）
- **功能**：識別成功關鍵因素、計算屬性權重
- **技術**：相關性分析、重要性排序

#### 5. 理想點分析模組（idealPointModule）
- **功能**：MAMBA方法實現、目標達成分析
- **技術**：理想點計算、差距分析

#### 6. 策略定位模組（strategyPositioningModule）
- **功能**：四象限矩陣分類、策略優先級建議
- **技術**：表現vs重要性矩陣、策略分類

#### 7. 品牌識別度模組（brandIdentityModule）
- **功能**：獨特性評估、差異化分析
- **技術**：Z-score標準化、獨特性指標

#### 8. AI洞察模組（aiInsightsModule）
- **功能**：綜合分析報告、策略建議
- **技術**：GPT-4o-mini深度分析、結構化報告

#### API調用封裝
```r
call_gpt_api <- function(prompt_id, variables = list(), max_tokens = 2048) {
  # 準備消息
  messages <- prepare_gpt_messages(prompt_id, variables, prompts_df)
  
  # API請求
  response <- httr2::request(api_url) %>%
    httr2::req_headers(
      "Authorization" = paste("Bearer", api_key),
      "Content-Type" = "application/json"
    ) %>%
    httr2::req_body_json(list(
      model = "gpt-4o-mini",
      messages = messages,
      temperature = 0.3,
      max_tokens = max_tokens
    )) %>%
    httr2::req_timeout(60) %>%
    httr2::req_perform()
  
  # 解析回應
  httr2::resp_body_json(response)$choices[[1]]$message$content
}
```

## 市場賽道分析深度解讀指南

### 理解成長潛力指標

成長潛力代表產品或品牌在特定屬性上的改善空間，計算方式為：
```
成長潛力 = (理想值 - 當前表現) / 理想值 × 100%
```

**解讀標準**：
- **80-100%**：極大改善空間
  - 含義：該屬性表現極差，有巨大提升潛力
  - 策略：若重要性高，應立即投入資源改善
  - 範例：如果「售後服務」成長潛力90%，表示目前服務水準極低

- **50-80%**：中等改善空間
  - 含義：該屬性表現一般，有明顯提升空間
  - 策略：制定改善計畫，分階段提升
  - 範例：如果「產品品質」成長潛力65%，需要品質改善專案

- **20-50%**：有限改善空間
  - 含義：該屬性表現尚可，但仍有改善餘地
  - 策略：評估投資報酬率，選擇性改善
  - 範例：如果「包裝設計」成長潛力35%，可考慮小幅優化

- **0-20%**：接近理想狀態
  - 含義：該屬性表現優秀，接近市場最佳水準
  - 策略：維持現狀，防止競爭對手超越
  - 範例：如果「價格競爭力」成長潛力10%，表示定價策略成功

### 理解市場重要性指標

市場重要性反映消費者對該屬性的重視程度，來源於：
- 關鍵因素分析的權重計算
- 消費者評論中的提及頻率
- 對整體滿意度的影響程度

**重要性等級**：
- **100%（關鍵屬性）**：
  - 這些是決定品牌成敗的核心屬性
  - 通常包括：品質、價格、核心功能
  - 必須優先確保這些屬性的競爭力

- **50%（重要屬性）**：
  - 對品牌表現有顯著影響的屬性
  - 包括：設計、服務、附加功能
  - 需要保持良好水準但非首要投資

### 四象限策略實務應用

#### 🟢 高潛力賽道（右上象限）
**特徵**：高成長潛力（>50%） + 高重要性（>75%）
**策略優先級**：最高
**行動方案**：
1. 立即制定改善計畫
2. 配置充足資源
3. 設定明確KPI追蹤進度
4. 考慮引入外部專家或技術

**實例**：某廚具品牌發現「安全性」位於此象限
- 現狀：安全性評分僅2.5/5
- 行動：重新設計產品、增加安全認證、強化品質控管
- 結果：6個月後提升至4.2/5，銷量增長35%

#### 🔵 成熟賽道（左上象限）
**特徵**：低成長潛力（<30%） + 高重要性（>75%）
**策略優先級**：維持
**行動方案**：
1. 持續監控競爭對手動態
2. 微調優化維持領先
3. 作為品牌優勢持續宣傳
4. 防禦性投資避免下滑

**實例**：某咖啡品牌發現「口感」位於此象限
- 現狀：口感評分4.5/5，已是市場領先
- 行動：維持品質標準、定期品質檢測
- 結果：穩固市場領導地位

#### 🟡 利基賽道（右下象限）
**特徵**：高成長潛力（>50%） + 低重要性（<50%）
**策略優先級**：差異化機會
**行動方案**：
1. 評估是否能創造獨特賣點
2. 小規模測試市場反應
3. 若成功可作為差異化策略
4. 控制投資規模

**實例**：某家電品牌發現「智慧連網」位於此象限
- 現狀：功能缺乏但市場需求不高
- 行動：開發基礎IoT功能，定位科技領先形象
- 結果：吸引科技愛好者族群，開拓新市場

#### ⚪ 低優先賽道（左下象限）
**特徵**：低成長潛力（<30%） + 低重要性（<50%）
**策略優先級**：最低
**行動方案**：
1. 維持基本水準即可
2. 不主動投資改善
3. 資源重新配置到高優先領域
4. 定期檢視是否需要調整

### 實戰案例分析

**案例：某運動鞋品牌的市場賽道分析結果**

前5大優先發展賽道：
1. **舒適度**：成長潛力72%，重要性100%
   - 診斷：產品舒適度明顯落後競爭對手
   - 策略：重新設計鞋墊、改善材質、增加尺寸選擇

2. **耐用性**：成長潛力68%，重要性95%
   - 診斷：產品壽命低於消費者預期
   - 策略：強化品質測試、升級材料、延長保固期

3. **外觀設計**：成長潛力55%，重要性85%
   - 診斷：設計過於保守，缺乏吸引力
   - 策略：聘請知名設計師、推出限量聯名款

4. **價格合理性**：成長潛力45%，重要性90%
   - 診斷：性價比略低於競爭對手
   - 策略：優化成本結構、推出不同價位產品線

5. **品牌形象**：成長潛力62%，重要性70%
   - 診斷：品牌認知度和好感度待提升
   - 策略：加強品牌故事傳播、贊助運動賽事

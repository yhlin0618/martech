## 🔍 分析 1：購買週期分群 (Lines 196-345)

### 分析目的

將客戶根據**購買間隔**分成三群：常買、正常、不常買

### 使用資料 (Lines 200-203)

```r
# 使用 ipt_mean (平均購買間隔)
# 只分析 ni >= 2 的客戶（至少回購過一次）
df_with_ipt <- df %>%
  filter(!is.na(ipt_mean), ni >= 2)
```

### 分群邏輯 (Lines 209-267)

**Step 1：計算分位數**

```r
p80 = quantile(ipt_mean, 0.8)  # 第 80 百分位
p20 = quantile(ipt_mean, 0.2)  # 第 20 百分位
```

**Step 2：分類（P20/P80 方法）**

```r
purchase_cycle_level = case_when(
  ipt_mean >= p80 ~ "高購買週期",   # 購買間隔長（不常買）
  ipt_mean >= p20 ~ "中購買週期",   # 正常頻率
  TRUE ~ "低購買週期"               # 購買間隔短（常買）
)
```

**特殊處理（Lines 216-266）**：

| 情況 | 處理方式 |
|------|----------|
| P20 = 最小值 | 強制最底部 20% 為「低」 |
| P80 = 最大值 | 強制最頂部 20% 為「高」 |
| 所有值太接近 | 強制均分成三等份 |

### 輸出統計 (Lines 270-278)

每一群統計：
- 客戶數量
- 佔比 (%)
- 平均購買週期（天）

### 視覺化 (Lines 289-345)

- **表格**：顯示三群的統計數據
- **圓餅圖**：視覺化客戶分布（紅/黃/綠配色）

## 🛒 分析 3：客單價分析 (Lines 507-646)

### 分析目的

### 使用資料 (Lines 512-520)

```r
# 計算 AOV (Average Order Value)
df <- df %>%
  mutate(aov = m_value / ni)  # 總消費 ÷ 交易次數

# 只分析新客和主力客
df_filtered <- df %>%
  filter(customer_dynamics %in% c("newbie", "active"))
```

### 統計邏輯 (Lines 527-542)

**分別統計新客和主力客**：

```r
group_by(customer_dynamics) %>%
summarise(
  customer_count = n(),           # 客戶數
  avg_aov = mean(aov),            # 平均客單價
  median_aov = median(aov),       # 中位數客單價
  total_sales = sum(m_value)      # 總銷售額
)
```

### 差異計算 (Lines 544-559)

# 計算主力客 vs 新客的差異
diff_amount = active_aov - newbie_aov

## 關鍵設計特點

### 1. 穩健的分群邏輯

- 所有分群都處理邊界情況
- 確保永遠產生三群（高/中/低）
- 避免因資料分布極端而分群失敗

### 2. 只分析有效資料

| 分析類型 | 過濾條件 |
|----------|----------|
| 購買週期 | ni ≥ 2（回購客戶） |
| 過去價值 | m_value > 0（有消費記錄） |
| 客單價 | 新客和主力客（有明確生命週期） |

### 3. 使用 80/20 法則

```
前 20% → 高
中間 60% → 中
後 20% → 低
```

這是經典的帕累托法則，適用於大多數業務場景。

## 實際應用範例

假設某電商有 5,000 位客戶：

### 購買週期分群結果

```
高購買週期（不常買）: 1,000 位 (20%) - 平均 60 天買一次
中購買週期（正常）:   3,000 位 (60%) - 平均 30 天買一次
低購買週期（常買）:   1,000 位 (20%) - 平均 10 天買一次
```

### 過去價值分群結果

```
高價值顧客: 1,000 位 (20%) - 平均消費 $5,000，總貢獻 $5,000,000 (71%)
中價值顧客: 3,000 位 (60%) - 平均消費 $1,200，總貢獻 $3,600,000 (26%)
低價值顧客: 1,000 位 (20%) - 平均消費 $300，總貢獻 $300,000 (3%)
```

**發現**：前 20% 客戶貢獻 71% 營收（符合 80/20 法則）

### 客單價分析結果

```
新客:    3,200 位 - 平均客單價 $450
主力客:    800 位 - 平均客單價 $650

差異: 主力客客單價比新客高 $200 (44%)
```

**洞察**：主力客單價明顯提升，表示培育策略有效！

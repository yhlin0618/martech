# BrandEdge Premium 使用指南

## 兩種資料上傳方式

### 方式一：上傳評論資料（傳統流程）
適用於：有原始評論資料，需要AI萃取屬性並評分

**步驟：**
1. 點擊側邊欄「1. 上傳評論資料」
2. 上傳包含以下欄位的檔案：
   - `Variation`：品牌名稱
   - `Title`：評論標題
   - `Body`：評論內容
3. 系統會透過AI萃取屬性（10-30個）
4. AI自動對每則評論進行屬性評分
5. 進行各項分析

### 方式二：上傳已評分資料（新功能）✨
適用於：已有評分完成的資料，想直接進行分析

**步驟：**
1. 點擊側邊欄「或 上傳已評分資料」
2. 上傳CSV或Excel檔案
3. 系統會自動識別欄位，您需要：
   - 選擇產品ID欄位（如：product_id、ASIN、SKU）
   - 選擇產品名稱欄位
   - 選擇品牌欄位（選填）
   - 勾選所有屬性評分欄位（數值型欄位）
4. 點擊「處理並預覽資料」
5. 確認後自動跳轉到分析頁面

## 資料格式範例

### 評論資料格式（方式一）
```csv
Variation,Title,Body
品牌A,很新鮮,產品很新鮮，味道很好，包裝也很用心...
品牌B,方便快速,配送速度快，產品品質不錯...
```

### 已評分資料格式（方式二）
```csv
product_id,product_name,brand,新鮮,美味,方便,營養,健康,安全,品質
PRD001,產品A,品牌A,8,7,9,6,7,8,9
PRD002,產品B,品牌B,7,8,8,7,8,9,8
```

## 關鍵差異

| 項目 | 傳統流程 | 新流程 |
|------|----------|--------|
| 輸入資料 | 評論文字 | 已評分數值 |
| 必要欄位 | Variation, Title, Body | 自由選擇 |
| AI處理 | 需要（萃取+評分） | 不需要 |
| 處理時間 | 較長 | 立即 |
| API成本 | 有 | 無 |

## 常見問題

### Q: 顯示「缺少必要欄位」錯誤？
A: 您可能使用了錯誤的上傳選項。
- 如果有評論資料 → 使用「1. 上傳評論資料」
- 如果有已評分資料 → 使用「或 上傳已評分資料」

### Q: 屬性欄位如何識別？
A: 系統會自動識別數值型欄位作為潛在屬性，您可以手動調整選擇。

### Q: 沒有品牌欄位怎麼辦？
A: 品牌欄位是選填的。如果沒有選擇，系統會使用預設品牌名稱。

### Q: 評分範圍應該是多少？
A: 建議使用 0-10 或 0-100 的範圍，系統會自動處理。

## 最佳實踐

1. **資料準備**：確保評分資料的完整性，減少缺失值
2. **屬性選擇**：選擇 10-30 個屬性以獲得最佳分析效果
3. **品牌識別**：如果有多品牌，請確保選擇品牌欄位
4. **資料品質**：上傳後檢查資料品質報告，確認無誤再進行分析

## 技術支援

如遇到問題，請檢查：
1. 檔案格式是否為 CSV 或 Excel
2. 是否選擇了正確的上傳選項
3. 資料是否包含必要的欄位
4. 數值欄位是否為有效數字

### 1. **KPI 定義提示** ✅
| KPI | 定義 | tooltip |

### 2. **新增客戶狀態方框** ✅
| 狀態 | 代碼 | 說明 | AI分析 |
|------|------|------|--------|
| 靜止戶預測 | - | 30天內預測可能沉睡客戶 | - |
| 首購客 | N | 首次購買的新客戶 | ✓ |
| 主力客 | E0 | 高價值活躍客戶 | - |
| 瞌睡客 | S1 | 購買頻率開始下降 | ✓ |
| 半睡客 | S2 | 接近流失邊緣 | - |
| 沉睡客 | S3 | 長期未購買 | ✓ |

```r

  bs4TabPanel(tabName = "留存分析", ...)
)

  tabPanel(title = "留存分析", ...)
)
```

# VitalSigns 旗艦版使用指南

## 概述

VitalSigns 旗艦版在原有的客戶 DNA 分析基礎上，新增了四大關鍵板塊，提供全方位的客戶洞察和行銷決策支援。本指南將協助您充分利用這些新功能。

## 系統流程

```
登入系統 → 上傳資料 → DNA 分析 → 四大板塊分析
```

## 快速開始

### 步驟 1：資料準備

支援的資料格式：
- Amazon 銷售報告（CSV/Excel）
- 一般交易記錄（需包含客戶ID、時間、金額）

必要欄位：
- **客戶識別**：Email 或客戶 ID
- **交易時間**：Purchase Date 或類似時間欄位
- **交易金額**：Item Price 或類似金額欄位

### 步驟 2：登入與上傳

1. 使用您的帳號密碼登入系統
2. 點擊「資料上傳」
3. 選擇一個或多個檔案（系統會自動合併）
4. 點擊「上傳並預覽」確認資料
5. 點擊「下一步」進行 DNA 分析

### 步驟 3：DNA 分析

1. 系統會自動開始分析（需要幾秒鐘）
2. 分析完成後可查看：
   - 視覺化圖表（ECDF、直方圖）
   - 詳細數據表
   - AI 分析洞察

### 步驟 4：四大板塊分析

完成 DNA 分析後，側邊欄會顯示四個新的分析板塊：

## 四大板塊詳解

### 1. 營收脈能 (Revenue Pulse) 💰

**目的**：了解收入結構和客戶價值分布

**關鍵指標**：
- **銷售額**：總收入規模
- **ARPU**：人均購買金額，評估客戶平均貢獻
- **客單價對比**：新客 vs 主力客的消費差異
- **CLV 分布**：客戶終生價值的分群狀況
- **交易穩定度**：客戶購買行為的規律性

**使用建議**：
- 觀察新客與主力客單價差異，調整定價策略
- 根據 CLV 分布，識別高價值客群
- 監控交易穩定度，預測收入波動

### 2. 客戶增長 (Customer Acquisition) 👥

**目的**：監測客戶基礎的擴張和結構變化

**關鍵指標**：
- **顧客總數**：當前活躍客戶規模
- **累積顧客數**：歷史總客戶數
- **新增率**：新客戶占比
- **客戶結構**：N/E0/S1-S3 狀態分布
- **獲客漏斗**：從潛在到忠誠的轉化過程

**使用建議**：
- 追蹤新客占比，評估獲客效果
- 分析客戶結構，了解健康度
- 優化漏斗轉化，提升客戶質量

### 3. 客戶留存 (Customer Retention) 🔒

**目的**：評估客戶忠誠度和流失風險

**關鍵指標**：
- **留存率**：持續活躍客戶比例
- **流失率**：沉睡客戶占比
- **風險客戶**：S1+S2 客戶數量
- **RFM 熱力圖**：多維度客戶分群
- **流失預測**：基於 AI 的風險評估

**使用建議**：
- 重點關注風險客戶（S1、S2）
- 使用 RFM 分析制定差異化策略
- 提前介入高流失風險客群

### 4. 活躍轉化 (Engagement Flow) 🔥

**目的**：優化客戶參與度和購買頻率

**關鍵指標**：
- **CAI 指數**：客戶活躍度綜合評分
- **再購率**：重複購買客戶比例
- **購買頻率**：平均購買次數
- **購買週期**：客戶回購間隔
- **忠誠階梯**：客戶成長路徑

**使用建議**：
- 根據 CAI 分群，制定活躍策略
- 縮短購買週期，提升頻率
- 設計忠誠度計畫，推動階梯攀升

## 實戰應用案例

### 案例 1：提升客單價

1. 查看「營收脈能」中的客單價分析
2. 發現新客單價僅為主力客的 60%
3. 策略：設計新客專屬組合優惠，提升首購金額

### 案例 2：喚醒沉睡客戶

1. 在「客戶留存」查看風險客戶數量
2. 發現 S1+S2 客戶占比達 30%
3. 策略：針對不同沉睡程度，設計階梯式喚醒方案

### 案例 3：提升購買頻率

1. 在「活躍轉化」分析購買週期
2. 發現平均購買間隔為 45 天
3. 策略：在第 40 天發送個人化提醒，縮短週期

## 數據解讀技巧

### NES 狀態說明
- **N (New)**：首購客，需要培養忠誠度
- **E0 (Existing)**：主力客，品牌價值貢獻者
- **S1 (Sleepy 1)**：輕度休眠，容易喚醒
- **S2 (Sleepy 2)**：中度休眠，需要刺激
- **S3 (Sleepy 3)**：深度休眠，挽回成本高

### 關鍵閾值參考
- **留存率 > 80%**：健康
- **新客占比 15-25%**：平衡
- **再購率 > 40%**：良好
- **CAI > 50%**：活躍

## 常見問題

**Q1：為什麼某些圖表顯示"數據不足"？**
A：部分指標需要客戶有多次交易記錄，建議累積更多數據後查看。

**Q2：如何下載分析結果？**
A：每個數據表都支援複製和匯出功能，可直接複製到 Excel。

**Q3：分析結果多久更新一次？**
A：每次上傳新數據並執行 DNA 分析後，所有板塊會自動更新。

**Q4：可以比較不同時期的數據嗎？**
A：目前版本專注於當前數據分析，時間序列比較功能將在後續版本推出。

## 最佳實踐

1. **定期更新數據**：建議每月更新一次，追蹤趨勢變化
2. **交叉分析**：結合多個板塊的洞察，制定綜合策略
3. **設定基準**：記錄關鍵指標，建立內部基準線
4. **行動導向**：每個洞察都應轉化為具體行動方案
5. **持續優化**：根據結果反饋，調整策略參數

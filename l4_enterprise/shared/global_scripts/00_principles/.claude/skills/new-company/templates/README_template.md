## 🚀 Quick Start

### 1. 設定 Rawdata 資料夾（必須）

```bash
# 選擇一種方式：

# Option A: Symlink 到現有 Dropbox 資料夾
ln -s /path/to/existing/dropbox/data data/local_data/rawdata_{COMPANY}

# Option B: 用 Finder 移動現有 Dropbox 資料夾到 data/local_data/

# Option C: 建立空資料夾（純 API 資料）
mkdir -p data/local_data/rawdata_{COMPANY}
```

### 2. 設定環境變數（必須）

```bash
# 複製模板
cp .env.template .env
chmod 644 .env

# 編輯 .env 填入實際憑證
```

**必填憑證（依啟用平台）：**

| Platform | 環境變數 |
|----------|----------|
| Cyberbiz | `CBZ_API_TOKEN`, `CBZ_SHOP_ID` |
| eBay | `EBY_SSH_HOST`, `EBY_SSH_USER`, `EBY_SSH_PASSWORD`, `EBY_SQL_*` |
| Amazon | `AMZ_ACCESS_KEY`, `AMZ_SECRET_KEY` |
| AI | `OPENAI_API_KEY` |

### 3. 啟用平台（必須）

編輯 `app_config.yaml`：

```yaml
# 取消註解你需要的平台
platform:
  - cbz    # Cyberbiz
  - eby    # eBay
  # - amz  # Amazon

# 同時啟用 pipeline
pipeline:
  platforms:
    cbz:
      enabled: true
    eby:
      enabled: true
```

### 4. 測試連線

```bash
# 測試資料庫連線
Rscript scripts/global_scripts/98_test/test_connections.R

# 測試腳本解析
Rscript -e '
source("scripts/global_scripts/04_utils/fn_resolve_script_path.R")
print(paste("Company:", get_company_from_config()))
'
```

### 5. 執行 ETL Pipeline

```bash
cd scripts/update_scripts

# 合併配置
make config-merge

# 執行 ETL
make run PLATFORM=cbz
```

### 6. 啟動應用程式

```bash
# 本地開發
Rscript app.R

# 或用 RStudio 開啟 {COMPANY}.Rproj
```

# AI 行銷平台 Wiki

這份 Wiki 面向客戶、營運團隊與非技術使用者，說明平台各模組在看什麼、怎麼操作，以及常見指標要怎麼解讀。

## 模組地圖

| 模組 | 適合回答的問題 | 入口頁面 |
|------|----------------|----------|
| TagPilot | 我的客戶是誰、價值如何、流失風險在哪、該怎麼分群與行動？ | [TagPilot](TagPilot) |
| VitalSigns | 我的營收、成長、留存、互動、地區市場現在健不健康？ | [VitalSigns](VitalSigns) |
| BrandEdge | 我的產品在市場上站在哪裡，與競品相比差在哪裡？ | [BrandEdge](BrandEdge) |
| InsightForge 360 | 哪些評價、時間因素、產品屬性真的會推動銷量？ | [InsightForge](InsightForge) |
| Report Center | 如何把多個模組的洞察整合成一份可閱讀、可下載的報告？ | [ReportCenter](ReportCenter) |

## 建議閱讀路線

1. 想先掌握整體營運狀況，從 [[VitalSigns]] 開始。
2. 想理解客群結構與精準行銷名單，接著看 [[TagPilot]]。
3. 想做競品比較或產品定位，查看 [[BrandEdge]]。
4. 想找出銷量驅動因子與可操作的優化方向，查看 [[InsightForge 360|InsightForge]]。
5. 需要把多模組內容整理成正式報告時，使用 [[Report Center|ReportCenter]]。

## 使用前先知道

1. 大多數分析都會跟著目前的「平台」「產品線」或「國家」篩選條件一起變動。
2. 各頁面的 AI 洞察、AI 分群命名與整合報告，都是根據你當下看到的資料生成。
3. 術語頁的公式已依照專案實作驗證；若看到頁面中的欄位名稱，代表它在系統內有對應計算邏輯。

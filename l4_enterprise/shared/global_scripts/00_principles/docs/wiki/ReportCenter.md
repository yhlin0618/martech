# Report Center

Report Center 是整合報告中心。它不是單一分析模組，而是把多個模組的結果整理成一份可預覽、可下載的整合報告。

## 會整合哪些內容

| 來源模組 | 典型內容 |
|----------|----------|
| VitalSigns | 營收、成長、留存、互動等整體健康度 |
| TagPilot | 客戶價值、流失、生命週期與行銷策略 |
| BrandEdge | 品牌定位、競爭分群、策略象限 |
| InsightForge 360 | 口碑、時間因素、產品屬性驅動因子 |

## 顯示內容

- 報告生成指引
- 報告生成中的進度提示
- Report Preview
- Download Report 按鈕

## 操作方式

1. 先確認你要用的模組都已切到正確的篩選條件。
2. 在左側按下「Generate Integrated Report」。
3. 系統會自動收集可用的模組資料，生成整合報告。
4. 生成完成後可先預覽，再下載報告。

## 看懂重點

這個模組回答的是：**如果要把多個分析頁面整理成一份可交付、可閱讀的完整報告，現在該怎麼做？**

## 什麼時候最適合用

- 要做週會、月會或老闆簡報時
- 需要把客群、營收、品牌定位與驅動因素放在同一份敘事裡時
- 想用單一輸出快速同步跨部門共識時

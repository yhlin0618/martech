# BrandEdge

BrandEdge 是品牌定位與競品比較模組。它把產品屬性表、品牌 DNA、競爭群聚、關鍵因素、理想基準與策略建議串在一起，適合用來回答「我在市場上站在哪裡」。

## Tab 一覽

| Tab | 主要用途 | 你會看到什麼 |
|-----|----------|--------------|
| position | 看產品屬性總表 | 品牌 / 商品篩選後的屬性比較表 |
| positionDNA | 看品牌 DNA 輪廓 | 多商品、多品牌的屬性折線圖 |
| positionMS | 看市場區隔與競爭群 | 競爭定位散點圖、群集表、AI 分群報告 |
| positionKFE | 看關鍵成功因素 | 關鍵因素列表、基準商品 |
| positionIdealRate | 看接近理想值的程度 | 依 key factors 排序的 Ideal Rate 表 |
| positionStrategy | 看單一商品的定位策略 | 四象限策略圖與 AI 建議 |

## position

### 顯示內容
- Brand 與 Item ID 篩選器
- Position Table
- 系統會自動移除全空列與資料太稀疏的欄位，讓表格更可讀

### 操作方式
1. 先鎖定產品線。
2. 再選品牌或商品，做橫向比較。
3. 找到差異大的屬性後，再切到後面的 DNA、KFE 或 Strategy tab 深挖。

### 看懂重點
這個 tab 回答的是：**不同商品在各項屬性上的原始表現差在哪裡？**

## positionDNA

### 顯示內容
- 多品牌 / 多商品的屬性折線圖
- 品牌與屬性篩選器
- 可調整是否一開始顯示全部品牌、線寬與標記大小

### 操作方式
1. 先挑幾個你要比較的品牌或商品。
2. 再挑重要屬性，只保留你關心的軸。
3. 觀察哪個商品在哪些軸特別突出或特別弱。

### 看懂重點
這個 tab 回答的是：**如果把產品屬性畫成一張輪廓圖，誰的強弱最明顯？**

## positionMS

### 顯示內容
- Competitive Positioning 視覺化散點圖
- Cluster table，列出各群集的品牌與商品
- AI 自動命名的群集名稱
- AI Market Segmentation Analysis Report

### 操作方式
1. 先看散點圖中商品如何自然聚成幾群。
2. 再看 cluster table，確認每一群的組成。
3. 最後看 AI 分群報告，把群集轉成可溝通的市場語言。

### 看懂重點
這個 tab 回答的是：**市場上有哪些競爭群？我的產品跟誰最像、離誰最遠？**

## positionKFE

### 顯示內容
- Key Factors 清單
- Benchmark Products by Factor
- 可調整敏感度、是否顯示 benchmark、最多顯示幾個 factor

### 操作方式
1. 先看哪些屬性被系統判定為 key factors。
2. 再看每個 factor 的 benchmark 商品是誰。
3. 若因素太多或太少，可調整敏感度重新檢視。

### 看懂重點
這個 tab 回答的是：**真正決定市場表現的關鍵因素是哪幾個？誰做得最好？**

## positionIdealRate

### 顯示內容
- Ideal Rate Analysis 表
- 系統會根據 Ideal row 與 key factors，計算每個商品達到理想條件的程度
- 可切換 factor 選取方法與顯示設定

### 操作方式
1. 先確認系統中的 Ideal 基準是否合理。
2. 再看哪些商品在最多 key factors 上達標。
3. 若要評估「最接近理想商品」而不是「最強競品」，這頁最有用。

### 看懂重點
這個 tab 回答的是：**如果用理想商品當標準，現在誰最接近它？**

## positionStrategy

### 顯示內容
- 單一商品的四象限策略圖
- 四個象限分別是訴求、改善、劣勢、改變
- AI Strategy Analysis Results

### 操作方式
1. 先選一個商品。
2. 看四象限中哪些屬性應該拿來主打，哪些應該補強。
3. 若需要對外提案或內部簡報，可直接參考 AI 策略建議。

### 看懂重點
這個 tab 回答的是：**對這個商品來說，該主打什麼、補什麼、改什麼？**

# AI MarTech - 智能行銷科技平台

> 基於 AI 技術的精準行銷解決方案，為不同規模企業提供客戶洞察、產品定位與行銷優化服務

## 🎯 平台概述

AI MarTech 是一個整合性的智能行銷科技平台，運用人工智慧技術幫助企業：

- **深度理解客戶**：透過 AI 分析客戶行為，建立精準的客戶DNA檔案
- **優化產品定位**：基於市場評論的 AI 分析，找到最佳產品定位策略  
- **精準行銷投放**：運用統計模型預測行銷效果，提升 ROI
- **數據驅動決策**：將複雜的客戶數據轉化為可執行的商業洞察

## 🚀 快速開始

### 1. 系統需求
- **R** (>= 4.0.0) 及相關套件
- **PostgreSQL** 資料庫（生產環境）
- **OpenAI API Key**（AI 分析功能）

### 2. 安裝部署
```bash
# 複製專案
git clone --recursive https://github.com/kiki830621/ai_martech.git
cd ai_martech

# 智能同步所有組件（支援 Claude Code）
./bash/subrepo_sync.sh

# 配置環境變數（複製範例檔案並填入實際值）
cp .env.example .env
```

### 3. 執行應用程式
```bash
# 進入應用程式目錄
cd l1_basic/positioning_app

# 啟動應用
Rscript app.R
```

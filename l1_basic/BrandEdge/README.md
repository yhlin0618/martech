# Positioning App

品牌定位分析應用程式，使用 AI 技術分析客戶評論並生成品牌定位洞察。

## 🔒 資料安全說明

本專案採用「零資料」版本控制策略：
- **無真實資料**：GitHub 上不包含任何真實的客戶資料或評論
- **需要設置**：首次使用需執行設置腳本生成測試環境
- **安全優先**：即使倉庫被盜用，也無法獲得任何敏感資訊

詳見 [DATA_SECURITY.md](DATA_SECURITY.md)

## 快速開始

### 1. 首次設置
```bash
# 執行設置腳本，創建本地測試環境
Rscript setup_local_data.R
```

這會創建：
- 目錄結構（data/, database/, cache/）
- 測試用資料庫（含測試帳號）
- 假的測試資料（供開發使用）
- 設定檔範本

### 2. 配置應用程式
```bash
# 複製設定檔範本
cp config_template.R config.R

# 編輯 config.R，填入必要的設定
```

### 3. 執行應用程式
```r
# 在 R 中執行
shiny::runApp("app.R")
```

使用測試帳號登入：
- Username: `test`
- Password: `test123`

```

├── app.R                    # 主應用程式
├── data/                    # 資料目錄（不含實際資料）
│   ├── sample/             # 範例資料位置
│   ├── test/               # 測試資料位置
```

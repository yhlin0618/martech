# Principle Documentation Archive

This archive contains deprecated, superseded, and historical principle documentation. 

## Directory Structure

- **`legacy_principles/`** - Deprecated principles with deprecation dates and reasons
- **`superseded_patterns/`** - Old implementation patterns that have been replaced
- **`migration_history/`** - Historical migration guides and transition documentation
- **`version_snapshots/`** - Point-in-time snapshots of principle versions for audit trails

## Important Notice

**These documents are ARCHIVED and should NOT be used for current development.**

All content here has been superseded by current principles in the active directories. These archives are preserved for:
- Historical reference
- Audit trails
- Understanding system evolution
- Migration context

## Accessing Current Principles

For current, active principles, please refer to:
- `/natural/en/part1_principles/` - Current English principles
- `/natural/zh/part1_principles/` - Current Chinese principles

## Archive Naming Convention

Archived files follow the pattern: `[PRINCIPLE_ID]_[deprecation_date]_legacy.qmd`

Example: `DEV_R032_20250828_four_part_legacy.qmd`

## Deprecation Metadata

Each archived file includes metadata about:
- Deprecation date
- Reason for deprecation
- Replacement principle reference
- Migration notes
